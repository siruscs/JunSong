# 社区行为准则

> **作者：** Genesis·峻松
> **更新日期：** 2026-06-24

本项目致力于打造一个开放、友好、互助的技术社区。参与本项目即表示你同意遵守以下行为准则。

---

## 我们的承诺

作为社区成员、贡献者和维护者，我们承诺：

- 无论年龄、体型、可见或不可见的残障、族裔、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、种族、宗教或性取向，对每个人都保持尊重与包容。
- 以建设性的态度交流，欢迎不同的观点与经验。
- 对错误保持耐心，将错误视为学习的机会。

## 我们的标准

### 鼓励的行为

- 使用友善和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

### 不可接受的行为

- 使用带有性暗示的言语或图像，以及不受欢迎的性关注
- 发表挑衅、侮辱、贬损性的评论，进行人身或政治攻击
- 公开或私下的骚扰行为
- 未经明确许可发布他人的私人信息（如住址、邮箱）
- 其他在专业环境中被认为不适当的行为

## 维护者责任

项目维护者有权利和责任：

- 删除、编辑或拒绝不符合本准则的评论、提交、代码、Issue 或其他贡献
- 暂时或永久封禁任何有不当、威胁、冒犯或有害行为的贡献者

## 适用范围

本准则适用于本项目所有公共空间（GitHub Issue、PR、Discussion、邮件列表等），也适用于在公共场合代表本项目时的行为。

##  enforcement

如果你遇到或目睹了不可接受的行为，请通过以下方式报告：

- GitHub Issue（标注 `[CODE_OF_CONDUCT]`）
- 或联系项目维护者

所有投诉将被及时、公正地审查和调查。维护者有义务对事件报告者保密。

## 参考

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 2.1 版。
